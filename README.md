# Diabetes Legends Foundation

Marketing website for the Diabetes Legends Foundation — a 501(c)(3) hosting youth basketball clinics, senior education, and an annual pickleball tournament for the diabetes community.

**Live site:** https://diabeteslegends.org

## Stack

Plain HTML / CSS / JS. Hosted on GitHub Pages. No build step.

## Local preview

```bash
python3 -m http.server 8000
```

Open http://localhost:8000

## Deploying changes

```bash
git add . && git commit -m "your message" && git push
```

GitHub Pages re-publishes within ~30 seconds.
